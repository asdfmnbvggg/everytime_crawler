# Everytime Crawler 

1. .env.sample 파일을 복제한 후, 복제한 파일 이름을 .env로 바꾼다.

2. .env에서 게시판 코드와 키워드 등 세부 사항들을 변경한다.

키워드를 여러 개 넣을 때는 쉼표로 구분한다.

'''
KEY_WORD=자취,원룸,기숙사
'''

이 경우 `자취` 키워드로 START_PAGE부터 END_PAGE까지 수집한 뒤,
이어서 `원룸`, `기숙사` 순서로 같은 범위를 수집한다.
결과 CSV는 하나의 파일에 이어서 저장된다.

3. everytime.py의 우클릭을 하여 상대 경로를 복사해준다.

4. 터미널을 새로 열어준다.

5. python을 친 후, 상대 경로를 붙여넣어 준다.

'''
python (상대 경로)
'''

6. 실행을 시켜준다.

7. 로그인 화면이 뜨면 수동으로 아이디와 비밀번호를 넣어준다.

8. 이후 다시 코드 창으로 돌아와 터미널에 enter를 쳐준다.

9. 수집되는 것을 보며 기다린다.
